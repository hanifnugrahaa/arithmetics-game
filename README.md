# arithmetics-game

A simple arithmetic game written in Java, developed as a college assignment.

## Features
- Addition and subtraction operations
- Score and lives system
- Simple terminal-based UI

## How to Run
1. Make sure you have Java installed.
2. Compile and run the game:

```bash
javac src/MidExam/ArithmeticsGame.java
java MidExam.ArithmaticsGame
